# Kecerdasan Buatan Lanjut - Final Project - Kelompok 4
Ini adalah Final Project dari tugas mata kuliah Kecerdasan Buatan Lanjut. dan ini adalah project yang dikolaborasi oleh mahasiswa Universitas Amikom Yogyakarta dan mahasiswa Universitas Perjuangan Tasikmalaya. Project ini berfokus pada penerapan metode **Convolutional Neural Network (CNN)** untuk mendeteksi jenis cuaca dari citra langit. Tujuannya adalah melatih model agar mampu mengenali kondisi cuaca seperti cerah, mendung, atau hujan berdasarkan gambar input.
